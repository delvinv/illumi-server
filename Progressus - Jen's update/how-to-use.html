How to Use Illumi

Once the physical artefact is installed and set-up onto the WiFi, little is required to run and use the artefact.
Here are a few things you need to know:

  1. The book will always be switched on so it can search for messages from the server.
  2. Illumi runs from a portable charger. When this runs out of battery, Illumi will turn off. During this time, the device will not be discoverable by the server and will not receive any messages.
  3. To avoid the battery running out of charge, the device should be plugged in using the retractable USB when it is not being used.
  4. If the battery does run out, once it is plugged in, the device will boot up and reload the programme automatically.
  5. The LED positioned on the top of the book will flash when a new message is received.
  6. Once a message is received, there will be on-screen instructions to guide you through the task.
  7. Once the task is complete, the book will return to its original state of looking for messages from the server.

It is recommended that when the book is not being used, it should be plugged in to a power source to avoid missing out on messages that may be sent when the device has run out of battery.
